| Từ chính         | Mục gốc                    | Cách dịch chốt                    | Ghi chú cách dùng / không dùng                                                                |
| ---------------- | -------------------------- | --------------------------------- | --------------------------------------------------------------------------------------------- |
| **argument**     | argument                   | **lập luận**                      | Dùng như thuật ngữ gốc cho _argument_; không đổi sang **biện giải**.                          |
|                  | argumentation              | **lập luận toán học**             | Khi cần nhấn mạnh tính quá trình hoặc thực hành, có thể viết **hoạt động lập luận toán học**. |
|                  | mathematical argument      | **lập luận toán học**             | Không đồng nhất với **chứng minh**.                                                           |
| **conjecture**   | conjecture                 | **phỏng đoán**                    | Không dùng **giả thuyết** làm cách dịch mặc định; khi cần có thể viết **mệnh đề phỏng đoán**. |
| **deduction**    | deduction                  | **suy diễn**                      | Không dùng **diễn dịch** làm thuật ngữ chốt.                                                  |
|                  | deductive                  | **suy diễn**                      | Giữ cùng gốc với _deduction_.                                                                 |
|                  | deductive argument         | **lập luận suy diễn**             | Tách với **luận suy diễn**: một bên là sản phẩm trình bày, một bên là kiểu hoạt động tư duy.  |
|                  | deductive reasoning        | **luận suy diễn**                 | Giữ đúng quy ước _reasoning = luận_.                                                          |
| **explain**      | explain                    | **giải thích**                    | Dùng cho việc làm sáng rõ; không nhập với **biện minh**.                                      |
|                  | explain their thinking     | **giải thích cách nghĩ của mình** | Ưu tiên cách này hơn **giải thích tư duy của mình**.                                          |
|                  | explain why                | **giải thích vì sao**             | Giữ cố định.                                                                                  |
|                  | explanation                | **lời giải thích**                | Tự nhiên hơn **sự giải thích** trong đa số ngữ cảnh.                                          |
| **generalize**   | generalize                 | **khái quát hóa**                 | Không cần thay bằng các cách dài như **rút ra tính tổng quát** nếu không thật cần.            |
|                  | generalization             | **sự khái quát hóa**              | Khi cần gọn hơn trong câu, có thể rút thành **khái quát hóa**.                                |
| **induction**    | induction                  | **quy nạp**                       | Khi là _mathematical induction_, dùng **quy nạp toán học**.                                   |
|                  | inductive                  | **quy nạp**                       | Đối ứng với **suy diễn**.                                                                     |
|                  | inductive argument         | **lập luận quy nạp**              | Dùng ổn định.                                                                                 |
|                  | inductive reasoning        | **luận quy nạp**                  | Cùng một dòng với **luận suy diễn**.                                                          |
| **inference**    | inference                  | **sự suy ra**                     | Không dùng **suy luận** làm cách dịch chốt, để tránh chồng lấn với _reasoning = luận_.        |
| **justify**      | justify                    | **biện minh**                     | Phải phân biệt với **chứng minh** và **giải thích**.                                          |
|                  | justification              | **sự biện minh**                  | Khi cần tự nhiên hơn, có thể viết **phần biện minh** hoặc **căn cứ biện minh**.               |
|                  | justify a claim            | **biện minh cho một khẳng định**  | Không đổi thành **giải thích một khẳng định**.                                                |
|                  | justify a solution         | **biện minh cho một lời giải**    | Giữ rõ chức năng bảo vệ bằng căn cứ.                                                          |
| **logic**        | logic                      | **luận lý**                       | Không lấy **logic** làm thuật ngữ chuẩn mặc định.                                             |
|                  | logic (field)              | **luận lý học**                   | Không dùng **logic học** làm chuẩn chính thức.                                                |
|                  | logical                    | **có tính luận lý**               | Dùng mặc định; tránh **logic** như tính từ mặc định.                                          |
|                  | logical argument           | **lập luận có tính luận lý**      | Không dùng **lập luận logic**.                                                                |
|                  | logical conclusion         | **kết luận có tính luận lý**      | Giữ ổn định.                                                                                  |
|                  | logical structure          | **cấu trúc luận lý**              | Đây là dạng rút gọn tốt của họ _logical_.                                                     |
|                  | logically                  | **về mặt luận lý**                | Chốt theo bản sau cùng; không dùng **một cách logic**.                                        |
| **proof**        | proof                      | **chứng minh**                    | Giữ riêng cho _proof_; không thay bằng **chứng tỏ** hay **biện minh**.                        |
| **reasoning**    | reasoning                  | **luận**                          | Đây là quy ước nền; không quay lại dùng **lý luận**, **suy luận** làm từ chốt.                |
|                  | formal reasoning           | **luận hình thức**                | Không đổi sang **suy luận hình thức** hoặc **lập luận hình thức** như thuật ngữ chốt.         |
|                  | mathematical reasoning     | **luận toán học**                 | Khi ngữ cảnh đã rõ là toán, có thể rút về **luận**.                                           |
|                  | reasoning and sense making | **luận và hiểu**                  | Giữ cố định trật tự này; không đảo thành **hiểu và luận**.                                    |
| **sense making** | sense making               | **hiểu**                          | Không dùng **tạo nghĩa**, **kiến tạo nghĩa**, **làm cho có nghĩa** làm thuật ngữ chốt.        |
|                  | foundation of sense making | **nền tảng của hiểu**             | Dùng ổn định; không rút thành **nền hiểu** hay **nền của hiểu**.                              |
| **valid**        | valid                      | **hợp lệ**                        | Không dùng **hợp lý** cho _valid_ trong ngữ cảnh luận lý.                                     |
|                  | validity                   | **tính hợp lệ**                   | Không dùng **tính đúng đắn** làm mặc định.                                                    |
