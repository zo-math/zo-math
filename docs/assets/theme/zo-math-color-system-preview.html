Color System & UI Preview – Primary #ef5350, Secondary #ffca28, với các accent hài hòa.

1) Thuật ngữ cơ bản

Primary color – màu chủ đạo (đỏ #ef5350) xuất hiện nhiều nhất: nút chính, link, điểm nhấn thương hiệu.
Secondary color – màu phụ (vàng #ffca28) dùng tiết chế: badge, nhấn nhỏ, highlight.
Accent colors – màu điểm nhấn thêm (xanh, indigo, teal) để cân bằng nóng/lạnh, tránh đơn điệu.
Complementary – màu đối nghịch trên vòng màu (đỏ ↔ xanh lam / xanh ngọc; vàng ↔ tím) → tương phản mạnh.
Analogous – màu liền kề (đỏ ↔ cam ↔ hồng; vàng ↔ cam ↔ xanh lá) → hài hòa êm mắt.
Shades & Tints – shades là màu đậm hơn (thêm đen), tints là màu nhạt hơn (thêm trắng) của cùng một màu.

7) Cách đưa vào Quarto

SCSS (sandstone.scss)

$primary:   #ef5350 !default;
$secondary: #ffca28 !default;

$body-bg:   #ffffff !default;
$body-color:#2c2c2c !default;

/* Màu trạng thái đề xuất mapping */
$info:      #42a5f5 !default;  /* blue */
$success:   #1de8b5 !default;  /* teal */
$warning:   #f9a825 !default;  /* shade vàng */
$danger:    #d32f2f !default;  /* shade đỏ */
